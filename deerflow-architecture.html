2Agent 执行循环（最核心）

用户发一条消息，Agent 循环执行：调 LLM → 判断是否用工具 → 执行工具 → 结果返回 LLM → 直到 LLM 说"完成"。这是整个系统的灵魂。

8Harness / App 分层原则

核心框架（Harness）和应用层（App）严格分离。Harness 可独立发布为 npm 包，App 依赖 Harness，Harness 永远不依赖 App。
